# MEMO - Static Website Project

The goal of this project is to practice building a clean, responsive static website from scratch. The core design is heavily inspired by the **Pinterest-style masonry grid view**, creating a visually engaging layout to showcase curated imagery.

---

## Objectives
* **Semantic HTML:** Writing clean, accessible, and structured code using modern HTML5 elements.
* **CSS Styling & Layouts:** Flexbox, Grid, and standard positioning techniques to create modern UI layouts.
* **Responsive Design:** Using media queries to ensure the website looks great on desktops, tablets, and mobile screens.
* **Web Best Practices:** Understanding file structuring, naming conventions (like BEM), and clean coding habits.

## Tech Stack Used
* **HTML5:** For page structure and content semantic mapping.
* **CSS3:** For custom styling, typography, variables, and animations.
* **Git/GitHub:** For version control and deployment.

---

## What I Learned So Far
* How to structure a multi-page navigation flow.
* Centering elements efficiently using CSS Flexbox.
* Implementing responsive breakpoints for mobile optimization.
